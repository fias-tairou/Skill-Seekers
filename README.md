## Basis Git Workflow


Een nieuwe branch maken        bijvoorbeeld:  feature/delete-favorite-club
```bash
git branch <soort>/<naam>
```

Voeg alle bewerkte files toe
```bash 
git add -A
```

de veranderingen comitten

```bash 
git commit -m "<bericht>"
```

pushen naar remote

```bash 
git push
```
